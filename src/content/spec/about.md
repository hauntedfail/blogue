# About me

2017年からプログラミングをはじめて、現在まで継続的に Open Source Software 活動に取り組んでいます。
特にセキュリティ領域に強い関心があり、2020年には国内で動画配信サービスを提供する企業に対して Penetration Test を実施し、当該サービスの脆弱性を報告しました。
実務経験としては Software Engineering が主ですが、攻撃者視点でセキュリティを意識した、堅牢なソフトウェア開発を心がけています。また、攻撃者視点での Code Review を任されることもあります。

GitHubでは[nonnil](https://github.com/nonnil)として活動していて、成果物は主に[hauntedfail](https://github.com/hauntedfail/)にホストしています。
連絡は[email](mailto:supp@haunted.fail)までお願いします。
